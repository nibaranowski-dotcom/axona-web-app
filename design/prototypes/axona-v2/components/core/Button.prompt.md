The Axona button — lime primary action, plus dark and ghost variants for secondary use. Use `primary` for the single key action per view; `dark` on light panels where lime would clash; `ghost` for tertiary.

```jsx
<Button variant="primary">Book a demo</Button>
<Button variant="ghost" size="sm">See what it does</Button>
```

Variants: `primary` (lime), `dark` (near-black), `ghost` (outline). Sizes: `sm` / `md` / `lg`.
