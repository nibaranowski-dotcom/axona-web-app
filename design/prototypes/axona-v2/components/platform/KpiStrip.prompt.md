The summary row that opens every platform module — equal cells, big tight number over a mono uppercase label, hairline dividers, no shadow. Keep it to 3–5 stats.

```jsx
<KpiStrip stats={[
  { value: "$65.8M", label: "Pipeline" },
  { value: "$19.6M", label: "Weighted" },
  { value: "34%",    label: "Win rate" },
  { value: "84d",    label: "Avg cycle" },
]} />
```

Numbers are specific and machine-reported (mono labels). Pair it directly under the module topbar, above the hero card.
