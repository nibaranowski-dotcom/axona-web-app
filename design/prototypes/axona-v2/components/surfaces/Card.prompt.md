The signature Axona feature card — warm-grey surface, tight title, muted body, and a skeleton "mock UI" footer that suggests a product screenshot. Pass `children` to drop in a real preview instead of the skeleton.

```jsx
<Card title="Procurement" body="Agents draft POs from live BOM demand." caption="PO-10482 · DRAFT" />
```

Props: `title`, `body`, `caption` (skeleton label), `children` (custom footer).
